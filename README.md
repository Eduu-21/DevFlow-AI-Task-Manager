DevFlow AI Task Manager

DevFlow AI Task Manager is a full-stack intelligent productivity web application designed to help users efficiently manage tasks with AI-powered assistance.

It combines a modern frontend, a scalable backend, and Python-based AI services to deliver smart task organization and workflow optimization.

Features

- Create, update, and delete tasks
- AI-powered task assistance
- Smart workflow suggestions
- Responsive and modern UI
- Full-stack architecture (Frontend + Backend + AI service)

Tech Stack

- Frontend: React (Vite), HTML, CSS, JavaScript  
- Backend: Node.js, Express  
- AI Service: Python (Flask / script-based service)  
- Package Manager: npm
